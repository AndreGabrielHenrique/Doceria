# Resumo do Projeto Doceria

O projeto **Doceria** é uma aplicação web construída com React e Vite. Apresenta uma interface modular composta por componentes reutilizáveis, hooks personalizados e estilos em SASS. As principais seções incluem: `Banner`, `BemVindo`, `Sobre`, `Variacao`, `Visual`, `Festa` e `Contato`.

- **Tecnologias:** React, Vite, ESLint
- **Execução local:** `npm install` e `npm run dev`
- **Build de produção:** `npm run build` e `npm run preview`

Este repositório organiza os componentes em `src/components/`, hooks em `src/hooks/` e estilos em `src/styles/`.
